## 搬运流程

1. 起运：
    - 四个抬高器把服务器顶起来
    - 下面垫上叉车托盘，叉子放到托盘下面
    - 放下抬高器/升高叉子，这样就抬起来了
2. 放到后备箱：
    - 便携式叉车最高升到 90cm（126cm的太大了不方便随车），坦克300后备箱离地高度是82cm，正好够得着（实在不够就在后备箱下面垫个大的板，叉车推到这个垫板上）
    - 叉车尽可能往里送，至少让服务器和下面的托盘有一部分进去
    - 在靠里面的两个角下面垫上轮子，从外面两个角往里面推直到完整进去
    - 取下里面那两个垫的轮子
3. 移出后备箱：
    - 在后排和后备箱外面，分别用四个抬高器把服务器托盘顶起来
    - 4个角垫上轮子，降下和移走抬高器使得轮子承力
    - 把叉车伸进后备箱，把服务器拉出来直到托盘底部完整移到叉子上（外面的轮子有可能先露出后备箱，可以接住移开）
    - 抬高叉子使其承力（不再使用轮子）
    - 移动叉车使其离开后备箱
4. 送到目的地：
    - 推叉车到目标地，降下叉子
    - 如果要移动服务器，就在下面垫上轮子（也可用抬高器辅助），如果不需要，就直接连托盘一起放下去
5. 叉车随车：
    - 用两个斜坡板搭在后备箱和地面，把叉车顺着板子推上车辆/拉下地面
